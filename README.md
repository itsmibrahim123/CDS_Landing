**OCR · FastAPI · Next.js · PostgreSQL · Docker · On-Premise**

Covenant Diligence Systems (CDS) is an on-premise legal and property 
diligence platform that transforms unstructured property documents — 
sale deeds, mutations, allotment letters — into structured, bank-ready 
dossiers. Built for banks, law firms, and financial institutions 
operating in Pakistan.

Automates the full pipeline: document ingestion → OCR with confidence 
scoring → classification → structured extraction → rules-based exception 
detection → CP generation → bank pack export (DOCX + PDF).

Role-based access control, immutable audit logs, and signed document 
links ensure compliance with banking and regulatory requirements.

Stack: Next.js · FastAPI · PostgreSQL · MinIO · Redis + Celery · 
Tesseract/PaddleOCR · Docker Compose
